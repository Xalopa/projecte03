# 📦 P03: Rèplica Web (WordPress en Local)

## 📝 Breu Descripció
Dins de l'escenari de simulació empresarial *"Sobreviure en una empresa IT"* a **EverPia**, l'equip de consultoria ha rebut l'encàrrec de replicar de forma fidedigna la web corporativa real **[davidperalvarez.com](https://davidperalvarez.com/)**. 

L'objectiu és analitzar-ne l'arquitectura de continguts, l'estructura SEO, la jerarquia de navegació i l'estil visual per aprendre bones pràctiques de desenvolupament, manteniment i disseny professional en WordPress des d'un entorn de proves local.

## ⚙️ Configuració Global del Lloc

Per garantir el rigor tècnic exigit pel client, s'han realitzat els següents ajustos interns a WordPress:
1. **Lectura:** Definició de la pàgina d'**Inici** com a pàgina frontal estàtica.
2. **Navegació:** Configuració personalitzada del menú principal i els submenús estructurats per nivells.
3. **Enllaços permanents (Permalinks):** Ajustats a mode `%postname%` (Nom de l'entrada) per mantenir les URL idèntiques a les del sitemap original.
4. **Paràmetres Generals:** Configuració correcta de l'idioma, la zona horària i el títol formal del lloc.
