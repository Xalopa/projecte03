# 🛠️ Tecnologies a Analitzar

---

# 🐧 LVM — Logical Volume Manager

## 📌 Què és?

LVM és el sistema estàndard de Linux per gestionar emmagatzematge de manera flexible i dinàmica.

Permet agrupar discos físics i crear volums lògics que poden expandir-se o reduir-se sense afectar el sistema.

---

## ✅ Característiques principals

- 🔄 Redimensionament dinàmic de particions
- 💾 Gestió flexible de discos
- 📈 Escalabilitat elevada
- ⚡ Integració nativa amb Linux
- 🛡️ Compatibilitat amb RAID i snapshots

---

## 🧩 Components principals

```text
Discos físics → Volume Groups → Logical Volumes
```

| Component | Funció |
|---|---|
| Physical Volume (PV) | Disc físic |
| Volume Group (VG) | Grup de discos |
| Logical Volume (LV) | Partició virtual |

---

# 🪟 Storage Spaces — Espais d’Emmagatzematge

## 📌 Què és?

Storage Spaces és la tecnologia de Microsoft que permet virtualitzar discs físics dins de Windows Server.

Agrupa diversos discos per crear espais d’emmagatzematge resilients i escalables.

---

## ✅ Característiques principals

- 🧩 Agrupació de discs físics
- 🛡️ Tolerància a errors
- 📈 Escalabilitat senzilla
- ⚙️ Administració integrada a Windows
- 🔄 Suport per mirroring i parity

---

## 🧩 Components principals

```text
Discos físics → Storage Pool → Virtual Disk
```

| Component | Funció |
|---|---|
| Physical Disk | Disc físic |
| Storage Pool | Grup de discs |
| Virtual Disk | Disc virtual |

---

# ⚖️ Semblances i Diferències

---

## ✅ Semblances

- 📈 Escalabilitat dinàmica
- 💾 Virtualització de discos
- 🛡️ Tolerància a errors
- 🔄 Gestió flexible d’espai
- 🏗️ Orientats a servidors empresarials

---

## ❌ Diferències principals

<div align="center">

| Aspecte | 🐧 LVM | 🪟 Storage Spaces |
|---|---|---|
| Sistema Operatiu | Linux | Windows Server |
| Interfície | CLI predominant | GUI + PowerShell |
| Cost | Gratuït | Requereix llicència Windows |
| Facilitat d’ús | Més tècnic | Més intuïtiu |
| Integració empresarial | Linux environments | Microsoft ecosystem |
